The "my_list" module
================

using my_list

import my_list from 1-my_list.py

       >>> my_list = __import__("1-my_list").MyList

Now use it:

    >>> my_list = MyList([1, 2,  3, 4])
    >>> my_list.print_sorted()
    [1, 2, 3, 4]

    >>> my_list = MyList([11, 0, -6, -1])
    >>> my_list.print_sorted()
    [-6, -1, 0, 11]

    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

    >>> my_list = MyList("str"])
    >>> my_list.print_sorted()
    ['r', 's', 't']
